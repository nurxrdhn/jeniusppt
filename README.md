# JeniusPPT Modern Full

Ekstrak ZIP ini, lalu copy semua isi folder hasil ekstrak ke folder project lama:

```text
C:\Users\User\Downloads\jeniusppt
```

Pilih Replace jika diminta.

Lalu jalankan:

```bash
cd /c/Users/User/Downloads/jeniusppt
npm install
npm run dev
```

Kalau sudah aman:

```bash
git add .
git commit -m "Replace with modern full JeniusPPT UI"
git push
```
